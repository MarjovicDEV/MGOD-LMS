# Learning Management System (LMS)

## Overview
The Learning Management System (LMS) is a comprehensive web application developed using CodeIgniter 4, designed to facilitate online education and training. It provides an intuitive platform for educators and learners to interact, manage courses, and track learning outcomes.

## Features
- **User Management:** Easy registration and login for students and instructors, along with role-based access control.
- **Course Management:** Create and manage courses with multimedia content, quizzes, and interactive assignments.
- **Progress Tracking:** Monitor student progress through dashboards and detailed reporting.
- **Communication Tools:** Built-in messaging and notification system to enhance communication between students and educators.
- **Responsive Design:** Fully responsive interface that works seamlessly across devices, ensuring accessible learning on-the-go.

## Technology Stack
- **Backend:** PHP with CodeIgniter 4 framework
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript, with Bootstrap for responsive design

## Getting Started
1. Clone the repository: `git clone https://github.com/MarjovicDEV/LMS-.git`
2. Set up the database and configure your `.env` file.
3. Run the application on your local server and start exploring the LMS features.
